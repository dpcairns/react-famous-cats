## Array `map`, `filter`, and `sort`

| method | new array?              | purpose                                      |
| ------ | ----------------------- | -------------------------------------------- |
| map    | Yes                     | Array to array transformation for every item |
| filter | Yes                     | New array only has "truthy" items            |
| sort   | No (returns same array) | sort existing array                          |
