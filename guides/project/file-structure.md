# File structure guide for labs

Below is a generic file/folder structure that can be used as a reference for managing projects.

```

├── assets

    ├── image.png

├── src

    ├── someHelperFunction.js

    ├── index.js

    ├── main.css

├── test

    ├── index.html

    ├── index.js

    ├── tests.js

├── .eslintrc

├── .gitignore

├── .travis.yml

├── index.html

└── README.md

```
