Markdown
===

> Write docs the developer way

## Headings

```md
# h1
## h2
### h3
```

## Bold and Italic

```md
**bold**
_italic_
```

## Lists

### Bullets

```md

* two

```

### Numbered

```md
1. one
1. two
1. three
```
